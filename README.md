# projetFinalFs
Projet final fullStack --> création d'un blog avec connexion à un compte et gestion d'article.

register_form => création d'un compte utilisateur 
printAllArticle => affichage de tous les articles contenue sur la bdd 
login_form => connexion au blog avec le compte utilisateur 
dashboard => affichage du compte utilisateur et du cookies 
             affichage de la liste de tous les articles appartenant à l'utilisateur connecté 
addArticles => création d'un nouvel article appartenat à l'utilisateur connecté 
updateArticle => possibilité de modifier n'importe quel articles de l'utilisateur connecté grace à l'id de l'article
